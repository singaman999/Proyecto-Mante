Guía de Comandos del Sistema de Red

1. Comandos Básicos

Linux - ls: Muestra el contenido de un directorio.
Linux - cd: Cambia el directorio actual.
Linux - mkdir: Crea un nuevo directorio.
Linux - rmdir: Elimina un directorio vacío.

Linux - ls

Descripción: Muestra el contenido de un directorio.

Ejemplo: ls -l muestra el contenido con detalles como permisos y tamaño.

Linux - cd

Descripción: Cambia el directorio actual.

Ejemplo: cd /home/usuario te lleva al directorio del usuario.

Linux - mkdir

Descripción: Crea un nuevo directorio.

Ejemplo: mkdir nueva_carpeta crea una carpeta llamada "nueva_carpeta".

Linux - rmdir

Descripción: Elimina un directorio vacío.

Ejemplo: rmdir carpeta_vacia elimina la carpeta si no tiene archivos dentro.
