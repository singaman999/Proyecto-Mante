Guía de Comandos del Sistema de Red

1. Comandos Básicos

Linux - ls: Muestra el contenido de un directorio.
×

Comando ls

Uso: ls [opciones] [directorio]

Ejemplo: ls -l /home/usuario

Linux - cd: Cambia el directorio actual.
×

Comando cd

Uso: cd [directorio]

Ejemplo: cd /var/www

Linux - mkdir: Crea un nuevo directorio.
×

Comando mkdir

Uso: mkdir [opciones] directorio

Ejemplo: mkdir -p proyecto/backup

Linux - rmdir: Elimina un directorio vacío.
×

Comando rmdir

Uso: rmdir [opciones] directorio

Ejemplo: rmdir temp

Linux - cp: Copia archivos.
×

Comando cp

Uso: cp [opciones] origen destino

Ejemplo: cp -r carpeta_origen/ carpeta_destino/

Linux - mv: Mueve archivos.
×

Comando mv

Uso: mv [opciones] origen destino

Ejemplo: mv archivo.txt /ruta/destino/

Linux - rm: Elimina archivos.
×

Comando rm

Uso: rm [opciones] archivo

Ejemplo: rm -rf directorio/

Linux - touch: Crea un archivo vacío.
×

Comando touch

Uso: touch [opciones] archivo

Ejemplo: touch nuevo_archivo.txt

Linux - nano: Abre un editor de texto simple.
×

Comando nano

Uso: nano [opciones] archivo

Ejemplo: nano configuracion.conf

Linux - grep: Busca una cadena de texto en un archivo.
×

Comando grep

Uso: grep [opciones] patrón [archivo...]

Ejemplo: grep -i "error" log.txt

Linux - sort: Ordena las líneas de un archivo.
×

Comando sort

Uso: sort [opciones] [archivo]

Ejemplo: sort -n archivo.txt

Linux - uniq: Elimina las líneas duplicadas de un archivo.
×

Comando uniq

Uso: uniq [opciones] [entrada [salida]]

Ejemplo: sort archivo.txt | uniq

Linux - head: Muestra las primeras líneas de un archivo.
×

Comando head

Uso: head [opciones] [archivo...]

Ejemplo: head -n 20 archivo.log

Linux - tail: Muestra las últimas líneas de un archivo.
×

Comando tail

Uso: tail [opciones] [archivo]

Ejemplo: tail -f /var/log/syslog

Windows - dir: Muestra el contenido de un directorio.
×

Comando dir

Uso: dir [unidad:][ruta][nombre] [/opciones]

Ejemplo: dir C:\Windows\*.exe

Windows - cd: Cambia el directorio actual.
×

Comando cd

Uso: cd [/d] [unidad:][ruta]

Ejemplo: cd C:\Program Files

Windows - md: Crea un nuevo directorio.
×

Comando md

Uso: md [unidad:]ruta

Ejemplo: md "C:\Nueva Carpeta"

Windows - rd: Elimina un directorio vacío.
×

Comando rd

Uso: rd [unidad:]ruta [/s] [/q]

Ejemplo: rd /s /q "C:\Carpeta a eliminar"

Windows - copy: Copia archivos.
×

Comando copy

Uso: copy [origen] [destino]

Ejemplo: copy *.txt C:\backup\

Windows - move: Mueve archivos.
×

Comando move

Uso: move [origen] [destino]

Ejemplo: move archivo.txt C:\destino\

Windows - del: Elimina archivos.
×

Comando del

Uso: del [nombre]

Ejemplo: del *.tmp /s /q

Windows - type: Muestra el contenido de un archivo.
×

Comando type

Uso: type [archivo]

Ejemplo: type config.ini

Windows - edit: Abre un editor de texto simple.
×

Comando edit

Uso: edit [archivo]

Ejemplo: edit script.bat

Windows - findstr: Busca una cadena de texto en un archivo.
×

Comando findstr

Uso: findstr [opciones] cadena [archivo...]

Ejemplo: findstr "error" *.log

Windows - sort: Ordena las líneas de un archivo.
×

Comando sort

Uso: sort [opciones] [archivo]

Ejemplo: sort /r datos.txt

Windows - more: Muestra el contenido de un archivo de forma paginada.
×

Comando more

Uso: more [archivo]

Ejemplo: more largo.txt

2. Comandos de Administración de Redes

Linux - ifconfig: Muestra la configuración de la interfaz de red.
×

Comando ifconfig

Uso: ifconfig [interfaz] [opciones]

Ejemplo: ifconfig eth0

Linux - route: Muestra la tabla de enrutamiento.
×

Comando route

Uso: route [opciones]

Ejemplo: route -n

Linux - ping: Comprueba la conectividad a un host específico.
×

Comando ping

Uso: ping [opciones] destino

Ejemplo: ping -c 4 google.com

Linux - traceroute: Muestra la ruta que toma un paquete de red para llegar a un host específico.
×

Comando traceroute

Uso: traceroute [opciones] host

Ejemplo: traceroute example.com

Linux - nslookup: Resuelve nombres de dominio a direcciones IP y viceversa.
×

Comando nslookup

Uso: nslookup [opciones] [nombre] [servidor]

Ejemplo: nslookup google.com

Linux - iptables: Configura un firewall.
×

Comando iptables

Uso: iptables [opciones] comando [cadenas] [parámetros]

Ejemplo: iptables -A INPUT -p tcp --dport 22 -j ACCEPT

Windows - ipconfig: Muestra la configuración de la interfaz de red.
×

Comando ipconfig

Uso: ipconfig [opciones]

Ejemplo: ipconfig /all

Windows - netstat: Muestra información sobre las conexiones de red activas.
×

Comando netstat

Uso: netstat [opciones]

Ejemplo: netstat -ano

Windows - ping: Comprueba la conectividad a un host específico.
×

Comando ping

Uso: ping [opciones] destino

Ejemplo: ping -t google.com

Windows - tracert: Muestra la ruta que toma un paquete de red para llegar a un host específico.
×

Comando tracert

Uso: tracert [opciones] destino

Ejemplo: tracert example.com

Windows - nslookup: Resuelve nombres de dominio a direcciones IP y viceversa.
×

Comando nslookup

Uso: nslookup [nombre] [servidor]

Ejemplo: nslookup google.com 8.8.8.8

Windows - netsh: Configura un firewall.
×

Comando netsh

Uso: netsh [contexto] [subcomando] [parámetros]

Ejemplo: netsh advfirewall set allprofiles state on
