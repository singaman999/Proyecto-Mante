Guía de Comandos del Sistema de Red

1. Comandos Básicos

Linux - ls: Muestra el contenido de un directorio.
Linux - cd: Cambia el directorio actual.
Linux - mkdir: Crea un nuevo directorio.
Linux - rmdir: Elimina un directorio vacío.

2. Comandos de Administración de Redes

Linux - ifconfig: Muestra la configuración de la interfaz de red.
Windows - ipconfig: Muestra la configuración de la interfaz de red.

3. Comandos de Diagnóstico

Linux - dmesg: Muestra los mensajes del kernel.
Windows - eventvwr: Muestra el Visor de eventos.

4. Comandos de Gestión de Archivos

Linux - cp: Copia archivos.
Windows - copy: Copia archivos.

5. Comandos de Administración de Usuarios y Grupos

Linux - useradd: Crea un nuevo usuario.
Windows - net user: Crea, modifica y elimina usuarios.
